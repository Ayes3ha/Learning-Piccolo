# Piccolo Docs

这个目录收纳仓库内自研引擎部分的阅读笔记和架构文档，不包含 `engine/3rdparty` 下第三方库自己的文档。

## Reading Protocol

读取顺序 MUST 遵循 `README.md -> <topic>/root.md -> 专题文档`。

- 先在本文件定位一级主题目录。
- 再进入对应 `root.md` 选择任务入口。
- 只有在任务已经定位后，才继续读取该目录下的专题文档。
